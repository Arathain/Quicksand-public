#mechanic 

Ranking is determined via a per-character Performance Point system; For every unique stage completed, a "performance point" metric is granted based off of stage difficulty and, well, performance - contributing to a final "PP" grade. This grade is then put into a cumulative pool of *all* player grades, and ranked top to bottom; afterwards, it is weighted based off of position on the list. For example, a top ranking of 302 points will be weighted at 100%; the next ranking, at 278 points, would be weighted at 97.5%; so on and so forth. The cumulative result of all the grades combined determines a player's total Performance Points; <u>fractional results are not showed in-game, but are used to determine ranking conflicts for same-score rankings, and then presented to show the marginal difference.</u> 

A 'global' account ranking is created by taking all character scores across the board, and applying the same ranking system to them.

This system is implemented to allow for players to have a "constant", unbroken progression metric, that does not go down, while still allowing for an overall ranking system to be used.
Performance Points are used as a metric to deepen character bonds, unlocking extra interactions and cosmetics.
